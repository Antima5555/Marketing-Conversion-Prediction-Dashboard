# 📊 Marketing Conversion Prediction Dashboard

This project focuses on analyzing digital marketing campaigns and predicting customer conversions using Machine Learning and Power BI.

---

## 🚀 Project Overview

The goal of this project is to:

- Analyze marketing campaign performance
- Predict whether a customer will convert or not
- Identify high-probability customers
- Optimize marketing spend based on insights

---

## 🧠 Machine Learning

Model used:
- Logistic Regression (with class balancing)

Steps:
- Data preprocessing (encoding + scaling)
- Train-test split
- Model training
- Prediction generation

Outputs:
- Prediction (0/1)
- Probability (likelihood of conversion)

---

## 📊 Power BI Dashboard

### 1️⃣ Marketing Performance Dashboard
- Total Customers
- Total Conversions
- Conversion Rate
- Total Ad Spend
- Conversion by Campaign Channel
- Conversion by Age Group
- Email Performance
- Campaign Type Analysis

---

### 2️⃣ Customer Prediction Dashboard
- Predicted Conversions
- High Probability Customers
- Avg Conversion Probability
- Probability by Age Group
- Probability vs Ad Spend (Scatter Plot)
- Prediction by Campaign Type
- Detailed Segment Table (Gender, Campaign, Probability)

---

## 📌 Key Insights

- Certain campaign channels (like PPC, Referral) perform better
- High probability customers can be targeted for better ROI
- Not all high spend campaigns give high conversion
- Segmentation helps in smarter marketing decisions

---

## 🛠️ Tech Stack

- Python (Pandas, Scikit-learn)
- Machine Learning (Logistic Regression)
- Power BI (Dashboard & Visualization)

---

## 🎯 Conclusion

This project demonstrates how combining Machine Learning with BI tools can help businesses:
- Predict customer behavior
- Improve conversion rates
- Optimize marketing spend
